01. About Me

I am a Fullstack Developer focused on building scalable, reliable, and impactful digital systems. My work spans across government, banking, and healthcare industries, where performance, security, and system reliability are critical.

I have contributed to several national-scale and enterprise systems, including the OSS RBA platform for the Ministry of Investment and Downstreaming (BKPM Indonesia), a Risk Management System for Bank Syariah Indonesia, the SIM Klinik (Clinic Information System), and the "Situation Room Presiden" Executive Dashboard under the Ministry of State Secretariat. These projects required strong backend architecture, structured database design, and seamless integration between multiple services and stakeholders.

On the backend, I design and develop RESTful APIs using Node.js (NestJS, ExpressJS) and Python (Django, Ninja API). I focus on modular architecture, clean code practices, and scalable database design using PostgreSQL and other relational databases. I also have experience with microservices-oriented systems, CI/CD pipelines, containerization, and cloud deployments.

On the frontend, I build responsive and interactive interfaces using React and Tailwind CSS, and I have experience developing mobile applications with React Native.

I enjoy solving complex technical problems from end to end starting from requirement analysis and system design to deployment and optimization. For me, software development is not just about writing code, but about delivering systems that are stable, maintainable, and meaningful for users.

I'm continuously learning, refining my architecture thinking, and striving to grow into a stronger system oriented engineer.
